# Convert merged GWAS data to CaLMR input format

Convert merged GWAS data to CaLMR input format

## Usage

``` r
convert_to_CaLMR_format(df, outcome_trait)
```

## Arguments

- df:

  Data frame of merged SNP-level data (output of `LD_clumped_SNPs`).

- outcome_trait:

  Character string identifying the outcome.
